# Protobuf Code-gen

Tutorial: https://github.com/dropbox/pb-jelly

1. Install the latest protobuf release binary: https://github.com/protocolbuffers/protobuf/releases
2. Make sure you can run `protoc` from the command line. `pb-jelly` uses this to generate the Rust code.
2. Go to `server/build/proto_generation`
3. Run `cargo build` to generate the Rust code
